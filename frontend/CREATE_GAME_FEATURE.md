# 創建麻將團局功能說明

## ✅ 已完成的功能

### 1. 完整的表單欄位

#### 基本資訊
- **團局類型**: 固定為 "臨時揪團"
- **團局標題**: 自訂標題
- **開始時間**: 日期時間選擇器
- **總人數**: 2-4 人（麻將標準）
- **金額/籌碼**: 例如 100/20
- **麻將規則**: 基本三將、台麻、港式、日麻、見花、其他（可自訂）

#### 地點資訊
- **地點名稱**: 場地名稱（例如：台北麻將館）
- **詳細地址**: 完整地址
- **GPS 座標**: 自動獲取或手動輸入

#### 場地特色（多選）
- 有冷氣
- 有停車位
- 近捷運
- 可吸菸
- 提供餐點
- 安靜環境

#### 玩家限制（多選）
- 新手友善
- 中級以上
- 高手限定
- 女性優先
- 學生限定
- 無限制

#### 其他
- **活動說明**: 詳細描述（支援 AI 增強）

## 🗺️ GPS 座標功能

### 方式 1: 使用目前位置
1. 點擊「使用目前位置」按鈕
2. 瀏覽器會請求位置權限
3. 自動獲取 GPS 座標
4. 自動反向地理編碼獲取地址

### 方式 2: 輸入地址自動解析
1. 在「詳細地址」欄位輸入完整地址
2. 系統會自動進行地理編碼
3. 1 秒後自動獲取該地址的 GPS 座標
4. 座標會顯示在表單中

### 技術實現
- **瀏覽器 Geolocation API**: 獲取當前位置
- **OpenStreetMap Nominatim API**: 
  - 反向地理編碼（座標 → 地址）
  - 地理編碼（地址 → 座標）
- **自動防抖**: 輸入地址後 1 秒自動解析

## ⚠️ 注意事項

### 1. 位置權限
- 使用「目前位置」功能需要用戶授予瀏覽器位置權限
- 如果用戶拒絕，會顯示錯誤訊息
- 可以改用手動輸入地址的方式

### 2. 座標驗證
- 提交表單前會檢查座標是否為 (0, 0)
- 如果座標無效，會提示用戶提供地址或使用定位

### 3. 點數扣除
- 創建團局需要 120 點數
- 如果點數不足，API 會返回錯誤

### 4. 地理編碼限制
- 使用 OpenStreetMap Nominatim API（免費）
- 有使用頻率限制（每秒 1 次請求）
- 建議輸入完整且正確的地址以提高準確度

## 🧪 測試建議

### 測試案例 1: 使用目前位置
1. 開啟創建團局頁面
2. 點擊「使用目前位置」
3. 允許瀏覽器存取位置
4. 確認座標和地址自動填入

### 測試案例 2: 輸入地址
1. 在「詳細地址」輸入：台北市信義區信義路五段7號
2. 等待 1 秒
3. 確認座標自動顯示

### 測試案例 3: 完整創建流程
1. 選擇麻將類型：基本三將
2. 輸入標題：週五晚上打麻將
3. 選擇時間：未來的日期時間
4. 設定人數：4
5. 輸入金額：100/20
6. 輸入地點名稱：台北麻將館
7. 輸入地址或使用定位
8. 選擇場地特色：有冷氣、有停車位
9. 選擇玩家限制：新手友善
10. 輸入活動說明
11. 點擊「LAUNCH EVENT」
12. 確認團局創建成功並跳轉到首頁
