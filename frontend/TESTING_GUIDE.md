# MahjongClub_App 測試指南

## 🧪 測試新功能

### 前置準備

1. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

2. **確認登入狀態**
   - 使用 APP 用戶登入
   - 或使用 LINE Bot 用戶登入

---

## 📋 測試清單

### 1. ✅ 管理團局功能 (ManageGame)

#### 測試步驟
1. 創建一個新團局
2. 使用另一個帳號報名該團局
3. 回到團主帳號
4. 進入"我的團局" → 點擊團局卡片
5. 在團局詳情頁點擊"管理團局"按鈕

#### 預期結果
- ✅ 顯示團局基本資訊
- ✅ 顯示待審核報名列表
- ✅ 可以接受報名
- ✅ 可以拒絕報名
- ✅ 可以取消團局
- ✅ 取消團局時顯示確認對話框

#### 測試案例
- [ ] 接受報名成功
- [ ] 拒絕報名成功
- [ ] 取消團局成功
- [ ] 非團主無法訪問 (顯示錯誤訊息)
- [ ] 操作後正確更新狀態

---

### 2. ⭐ 評分團局功能 (RateGame)

#### 測試步驟
1. 參與一個團局 (或創建並接受報名)
2. 進入"我的團局"
3. 點擊團局卡片上的"評分"按鈕
4. 或在團局詳情頁點擊"評分團局"按鈕

#### 預期結果
- ✅ 顯示團局資訊
- ✅ 列出所有需要評分的玩家 (不包含自己)
- ✅ 可以選擇好評或差評
- ✅ 可以輸入評論
- ✅ 評論少於 5 個字時無法提交
- ✅ 顯示已評分的玩家狀態
- ✅ 批量提交所有評分

#### 測試案例
- [ ] 選擇好評並輸入評論
- [ ] 選擇差評並輸入評論
- [ ] 評論少於 5 個字時顯示錯誤
- [ ] 未選擇評分時無法提交
- [ ] 已評分的玩家顯示綠色標記
- [ ] 提交成功後導航回"我的團局"
- [ ] 重新進入頁面時顯示已評分狀態

---

### 3. 👤 查看用戶評價功能 (UserReviews)

#### 測試步驟
1. 進入"個人資料"頁面
2. 點擊"查看我的評價"按鈕
3. 或直接訪問 `/reviews/:userId`

#### 預期結果
- ✅ 顯示用戶基本資訊
- ✅ 顯示評價統計 (總評價、好評、差評)
- ✅ 顯示好評率進度條
- ✅ 列出所有評論
- ✅ 顯示評論時間

#### 測試案例
- [ ] 正確顯示用戶資訊
- [ ] 正確計算評價統計
- [ ] 好評率進度條正確顯示
- [ ] 評論列表正確排序
- [ ] 沒有評論時顯示空狀態

---

### 4. 💬 評分單一用戶功能 (RateUser)

#### 測試步驟
1. 訪問 `/rate-user?gameId=xxx&toUserId=xxx`
2. 選擇好評或差評
3. 輸入評論 (至少 5 個字)
4. 提交評分

#### 預期結果
- ✅ 顯示被評分用戶資訊
- ✅ 顯示團局資訊
- ✅ 可以選擇好評或差評
- ✅ 可以輸入評論
- ✅ 評論少於 5 個字時無法提交
- ✅ 提交成功後返回上一頁

#### 測試案例
- [ ] 正確顯示用戶和團局資訊
- [ ] 選擇好評並提交
- [ ] 選擇差評並提交
- [ ] 評論少於 5 個字時無法提交
- [ ] 提交成功後顯示成功訊息
